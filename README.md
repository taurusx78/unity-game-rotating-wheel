#Rotatine-Wheel Game
UNITY를 이용해 만든 캐주얼 게임이다.
<br>

###플레이방식
- 휠의 구멍을 피해 왼쪽 또는 오른쪽으로 이동한다.
- 시간이 지날수록 휠 회전 속도는 점점 빨라진다.
- 1초에 1점씩 점수가 쌓이면 300초를 버티면 게임에서 이기게 된다.
<br>
